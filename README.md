# How to set Virtual Env. 
## To create virtual env 
```commandline
python -m venv .venv  
```
## To start/Activate Venv

```commandline
cd venv/Scripts
activate
cd ..
cd ..
```

## To Stop/Deactivate Venv

```commandline
deactivate
```
## To create requirements.txt 

```commandline
pip freeze > requirements.txt
```
## Install Dependencies 

```commandline
pip install -r requirements.txt
```
